# schetti.com

Moderner zweisprachiger One-Page-Auftritt für **Olga Schetti – IT Engineering & Consulting**.

## Veröffentlichung über GitHub Pages

1. Alle Dateien aus diesem Ordner direkt in das Stammverzeichnis des Repositorys hochladen.
2. In GitHub unter **Settings → Pages** auswählen:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/ (root)`
3. Als Custom Domain `schetti.com` verwenden.
4. Nach erfolgreicher DNS-Prüfung `Enforce HTTPS` aktivieren.

## Dateien

- `index.html` – Startseite DE/RU
- `styles.css` – responsives Design, keine externen Fonts
- `script.js` – Sprachumschaltung und dezente Animationen
- `impressum.html` – Impressum DE/RU
- `datenschutz.html` – Datenschutzerklärung DE/RU
- `CNAME` – Custom Domain für GitHub Pages
- `favicon.svg`, `robots.txt`, `sitemap.xml`, `404.html`

## Hinweise

- Keine Cookies, Analytics, externen Fonts oder externen Skripte.
- Die Sprachwahl wird nur lokal im Browser gespeichert.
- Rechtstexte prüfen und aktualisieren, sobald weitere Funktionen, Formulare, Analyse- oder Drittanbieter-Dienste eingebaut werden.
